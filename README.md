# esp-open-sdk-vagrant

## What is it?

Vagrant provisioning scripts to create a ready-to-go VM for ESP8266 development using pfalcon/esp-open-sdk

## Prerequisites

1. Install VirtualBox
2. Install Vagrant

## Usage

```
  vagrant up
```
